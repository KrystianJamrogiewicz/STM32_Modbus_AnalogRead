# Analog measurement of temperature, voltage and current using STM32 BluePill and data transmission to PC via Modbus

Used components: 
- STM32F103C8T6 BluePill board 1
- W5500 Ethernet module
- 16-bit 4-channel ADC converter ADS1115
- 24-bit 4-channel ADC converter ADS1220
- PT1000 4 wires
- ACS712 current sensor module up to 5A
